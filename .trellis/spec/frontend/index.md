# Frontend Development Guidelines

> Best practices for frontend development in this project.

---

## Overview

Chip Solver is a single-page React 19 + TypeScript app built with Vite. It uses Tailwind CSS (dark neon theme), Framer Motion for animations, and Context + useReducer for state management. Deployed to GitHub Pages.

**Tech stack:** React 19, TypeScript 5.8, Vite 6, Tailwind CSS 3, Framer Motion, lucide-react, clsx, nanoid
